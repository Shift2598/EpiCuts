# EpiCuts - Barbershop Website

## How to Run

1. Clone the repo
   ```
   git clone <repo-url>
   cd EpiCuts
   ```

2. Install backend dependencies
   ```
   cd backend
   npm install
   ```

3. Start the server
   ```
   node server.js
   ```

4. Open browser
   ```
   http://localhost:3000
   ```

## Admin Panel
```
http://localhost:3000/admin
```
Login: `admin` / `admin123`

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: SQLite
